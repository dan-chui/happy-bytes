This is list of helpful links for HTML and CSS resources:

## HTML

[Can I Use](https://caniuse.com/) - helps check the compatiability of features with different browsers easily.

---

[CodePen](https://codepen.io/) - helps you to test out front-end code before you make any commitments.

---

[Favicon Generator](https://favicon.io/) - a free favicon generator - useful for your different projects.

---

## CSS

[Adobe Color](https://color.adobe.com/explore) - use Adobe's color wheel to create different color palettes.

---

[ColorHunt](https://colorhunt.co/) - a collection of user-created color palette schemes.

---

[CSS Animate](https://animate.style/) - easy to use CSS animations.

---

[Font Awesome](https://fontawesome.com/) - a powerful library of icons for many different purposes.

---

[UnDraw](https://undraw.co/) - open-source graphics and images.

---

[Hero Patterns](https://heropatterns.com/) - a collection of repeatable SVG background patterns.

---

[uiGradients](https://uigradients.com/#ByDesign) - a collection user-created gradient backgrounds.

---

[LottieFiles](https://lottiefiles.com/) - access to a library of lightweight animation files.

---

[Unsplash](https://unsplash.com/) - a collection of Royalty-free & Paid high-resolution images.

---

[Pixabay](https://pixabay.com) - access to both free images and videos.

---

[Pexels](https://pexels.com) - access to both free images and videos.
